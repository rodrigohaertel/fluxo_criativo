# Vídeo Avançado — Base de Conhecimento

Referência técnica e criativa para os três formatos avançados de vídeo: Cenas Reais, Motion/Animação e Híbrido.

---

## Formato 1 — Cenas Reais (Stock Footage + Filmagem Própria)

### O que é

Vídeo que usa filmagem real — seja gravada pelo próprio produtor ou montada com cenas de bancos gratuitos (stock footage). O script narra por cima das imagens. O resultado parece mais autêntico que avatar e tem maior potencial de engajamento orgânico.

### Quando usar

- Descoberta: vídeos que precisam parecer conteúdo orgânico, não anúncio
- Prova Social: mostrar resultados reais, ambiente real, pessoa real
- Educativo: demonstração prática, passo a passo visual
- Quando o produtor não quer aparecer com avatar, mas quer vídeo com narração

### Estrutura do Script com Cenas Reais

O script é dividido em **blocos de narração + cena visual correspondente**. O narrador fala em off (ou no próprio vídeo) enquanto as cenas aparecem.

Formato obrigatório de cada bloco:

```
[NARRAÇÃO 0-5s]
Texto da fala que será narrado neste trecho.

[CENA]
Descrição visual concreta do que aparece na tela durante esta narração.
Plano: [close / médio / aberto]
Ação: [o que acontece]
Mood: [tenso / calmo / inspirador / íntimo / enérgico]
```

### Estrutura da Shot List

Entregar sempre como tabela para facilitar a produção ou busca de stock:

| # | Plano | Ângulo | Ação descrita | Duração | Mood | Keywords em inglês |
|---|---|---|---|---|---|---|
| 1 | Close | Frontal | Pessoa olhando para o celular com expressão de preocupação | 3s | tenso | "person worried looking phone", "woman stressed mobile screen" |
| 2 | Médio | Lateral | Mãos digitando rapidamente no teclado | 4s | urgente | "hands typing keyboard fast", "person working laptop urgent" |
| 3 | Aberto | Plongée | Pessoa sozinha em ambiente caótico (mesa bagunçada) | 5s | sobrecarregado | "overwhelmed desk messy", "cluttered workspace chaos" |
| 4 | Close | Frontal | Sorriso genuíno ao ver resultado na tela | 3s | aliviado | "person smiling relief screen", "woman happy looking monitor" |

### Regras de Edição para Cenas Reais

1. **Corte no ritmo da narração**: cortar imagens no final de frases, nunca no meio
2. **Variedade de planos**: alternar close, médio e aberto para manter ritmo visual
3. **Duração máxima por cena**: 5-7 segundos antes de cortar para outra cena (evitar estagnação)
4. **Coerência de paleta**: se for stock, priorizar cenas com paleta de cor similar entre elas (mesma temperatura de luz)
5. **Audio**: narração em over (voz por cima), música de fundo em 15-20% do volume
6. **Legenda obrigatória**: 85% dos vídeos são assistidos sem som

### Dicas para Filmagem Própria

- **Luz natural**: filmar perto de janela, de costas para a luz. Evitar luz direta no rosto
- **Estabilização**: celular no tripé ou apoiado. Vídeo tremido afasta
- **Fundo limpo**: parede lisa, prateleira organizada, ou fundo com planta. Evitar bagunça
- **Roupas sólidas**: evitar listras e xadrez (vibram na câmera)
- **Microfone**: mesmo o de fone de ouvido simples é melhor que o microfone do celular sozinho

## Formato 2 — Motion / Animação

### O que é

Vídeo criado com elementos animados — textos, ícones, dados, gráficos em movimento. Não exige aparecer em câmera. Ótimo para explicar conceitos, mostrar números, criar conteúdo educativo de alto valor percebido.

### Quando usar

- Explicar um método (Furadeira, etapas de resultado)
- Mostrar dados ou resultados com impacto visual
- Conteúdo educativo com diagrama, checklist animado, countdown
- Quando o produtor não quer aparecer e o avatar não combina com o estilo do nicho

### Ferramentas e Perfil de Cada Uma

| Ferramenta | Perfil | Gratuito? | Melhor para |
|---|---|---|---|
| **Remotion** | Código React, controle total | Sim (open source) | Programadores, vídeos complexos com dados |
| **CapCut** | App mobile e desktop, templates prontos | Sim (plano gratuito) | Qualquer produtor, edição rápida, Reels |
| **Canva Animado** | Templates de apresentação/vídeo | Sim (plano gratuito) | Conteúdo educativo, carrossel animado |
| **Adobe Express** | Templates profissionais, export em vídeo | Sim (versão free limitada) | Criativos de marca, anúncios estáticos animados |

### Animation Brief — Estrutura Padrão

Tabela de componentes para entregar ao editor ou usar no Remotion:

| # | Componente | Entrada | Duração na tela | Saída | Conteúdo | Observações |
|---|---|---|---|---|---|---|
| 1 | Tela de abertura | Fade in | 2s | Slide left | Headline principal | Fonte bold, cor primária do produto |
| 2 | Dado de impacto | Scale up 0.8x-1x | 3s | Fade out | "10.000 alunos em 12 meses" | Número em destaque, cor de acento |
| 3 | Etapa 1 do método | Slide right | 4s | Permanece + próximo entra | Ícone + texto da etapa | Ícone simples, 1 linha de texto |
| 4 | Etapa 2 do método | Slide right | 4s | Permanece + próximo entra | Ícone + texto da etapa | Sequência visual de progressão |
| 5 | Etapa 3 do método | Slide right | 4s | Permanece | Ícone + texto da etapa | Fechamento da sequência |
| 6 | CTA final | Zoom in suave | 5s | Loop ou permanece | Chamada para ação + botão | Cor de contraste alto no CTA |

### Especificações Remotion

```javascript
// Parâmetros base para vídeo de 30s a 30fps
const FPS = 30;
const DURATION_IN_FRAMES = 900; // 30s x 30fps

// Dimensão padrão (Reels/Stories vertical)
const WIDTH = 1080;
const HEIGHT = 1920;

// Para YouTube/horizontal
const WIDTH_YT = 1920;
const HEIGHT_YT = 1080;

// Props típicos por componente
interface SlideProps {
  text: string;           // conteúdo textual
  delay: number;          // frame de entrada
  duration: number;       // frames visível
  color: string;          // HEX da cor
  animation: 'fadeIn' | 'slideRight' | 'scaleUp' | 'slideBottom';
}
```

Biblioteca recomendada para animações prontas no Remotion: `@remotion/transitions` e `remotion-animated`.

### Especificações CapCut (modo desktop)

1. Criar projeto: 1080x1920 (vertical) ou 1920x1080 (horizontal)
2. Adicionar clipes de cor sólida como fundo de cada "cena"
3. Adicionar texto com as templates de texto animado (seção "Texto" > "Animação")
4. Usar transições em 0.3s entre cenas (mais rápido que o padrão parece mais profissional)
5. Exportar em 1080p, H.264

### Especificações Canva Animado

1. Criar design novo: "Vídeo para Reels" (1080x1920) ou buscar template animado
2. Cada "slide" do Canva vira uma cena do vídeo
3. Aplicar animações de entrada e saída em cada elemento (painel "Animar")
4. Ajustar timing de cada slide para 2-4 segundos
5. Exportar como MP4

### Paleta e Tipografia para Motion

A paleta deve vir do perfil do produto. Se não definida, usar:
- **Fundo:** cor neutra escura (`#0a0a0f`) ou clara (`#f8f8f8`)
- **Primária:** cor de destaque do produto
- **Texto principal:** branco ou preto de alto contraste
- **CTA:** cor de acento de alto contraste com o fundo

Tipografias recomendadas para motion (disponíveis no Google Fonts):
- **Inter** (peso 700/900) — moderna, legível, tech
- **Montserrat** (peso 700/800) — energética, impactante
- **Lato** (peso 700) — clean, educacional
- **Poppins** (peso 600/700) — amigável, coaching

## Formato 3 — Híbrido (Avatar + B-roll)

### O que é

Combina segmentos de avatar IA (pessoa falando) com segmentos de b-roll (cenas externas, stock footage, demonstrações). O resultado é mais dinâmico visualmente que o avatar puro e mais barato de produzir que o cenas reais puro.

### Quando usar

- Vídeos de 30-90s que precisam de variedade visual
- Anúncios em vídeo que combinam autoridade (rosto falando) com prova visual (resultado, cena, dado)
- Conteúdo educativo onde parte da explicação precisa de demonstração visual

### Proporções Recomendadas por Objetivo

| Objetivo | Proporção | Lógica |
|---|---|---|
| **Descoberta** | 30% avatar / 70% b-roll | Parece mais conteúdo orgânico que anúncio. B-roll diversificado mantém atenção |
| **Relacionamento** | 50% avatar / 50% b-roll | Equilíbrio entre conexão humana (avatar) e variedade visual (b-roll) |
| **Conversão** | 70% avatar / 30% b-roll | Rosto cria confiança. B-roll reforça prova social ou resultado |
| **Prova Social** | 40% avatar / 60% b-roll | Avatar apresenta a prova, b-roll mostra o resultado real |
| **Tutorial/Passo a Passo** | 20% avatar / 80% b-roll | Avatar dá contexto, b-roll demonstra cada etapa |

### Estrutura do Script Segmentado

O script deve indicar claramente cada tipo de segmento com marcações:

```
[AVATAR 0-5s]
Fala completa do avatar neste trecho. Linguagem natural, frases curtas.
Expressão: [ex: sério, confiante, empático]

[B-ROLL 5-12s — NARRAÇÃO EM OFF]
O avatar não aparece. Narração continua por cima das cenas.
Cena: descrição concreta do visual
Plano: close / médio / aberto
Keywords stock: "palavras para buscar"

[RETORNO AVATAR 12-18s]
O avatar volta. Transição deve parecer natural — avatar continua o raciocínio.
Fala: texto da fala

[B-ROLL 18-25s — SEM NARRAÇÃO]
Cena de impacto visual ou dado animado. Sem voz, só música de fundo.
Texto na tela: [headline de reforço que aparece na cena]
Cena: descrição

[AVATAR 25-35s — CTA FINAL]
Avatar finaliza com chamada para ação direta.
```

### Regras de Coerência Visual Avatar + B-roll

1. **Temperatura de cor consistente**: se o avatar tiver fundo quente (bege, creme), buscar b-roll com paleta quente também. Câmbios bruscos de cor "denunciam" a montagem.
2. **Corte limpo**: cortar do avatar no início da frase de narração do b-roll, nunca no meio de uma palavra
3. **Texto na tela no b-roll**: quando não há fala, adicionar texto na tela para âncora visual (85% assiste sem som)
4. **B-roll não pode competir com a narração**: evitar cenas com muita ação ou movimento quando a narração está explicando algo complexo
5. **Duração de b-roll**: cada clip de b-roll não deve exceder 7-8 segundos antes de cortar ou retornar ao avatar

### Configuração HeyGen para Vídeo Híbrido

Ao gerar com HeyGen, o avatar precisa ter **fundo removido ou fundo neutro** para facilitar a composição com b-roll no editor:

```json
{
  "background": {
    "type": "color",
    "value": "#00FF00"
  }
}
```

Usar fundo verde (`#00FF00`) para permitir chroma key no editor de vídeo.

Alternativa: fundo branco ou preto sólido e usar o modo "subtract" no editor.

### Ferramentas de Edição para Híbrido

| Ferramenta | Nível | Custo |
|---|---|---|
| **CapCut** | Iniciante | Gratuito |
| **DaVinci Resolve** | Intermediário | Gratuito |
| **Adobe Premiere** | Avançado | Pago |
| **Final Cut Pro** | Avançado (Mac) | Pago |

Para a maioria dos produtores digitais, **CapCut** é suficiente para editar híbridos de 30-90s.
