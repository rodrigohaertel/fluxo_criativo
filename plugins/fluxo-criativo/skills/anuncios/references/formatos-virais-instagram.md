# Formatos de Imagem Estatica Mais Virais no Instagram. Referencia para Anuncios

Base de conhecimento com os formatos visuais que mais convertem em anuncios estaticos no Instagram. Atualizada com dados de 2025-2026.

---

## Dimensoes e Specs por Posicionamento

| Posicionamento | Dimensao (px) | Proporcao | Formato |
|---|---|---|---|
| Feed (padrao) | 1080 x 1350 | 4:5 (retrato) | JPG/PNG |
| Feed (quadrado) | 1080 x 1080 | 1:1 | JPG/PNG |
| Stories / Reels cover | 1080 x 1920 | 9:16 (vertical) | JPG/PNG |
| Carrossel | 1080 x 1080 | 1:1 por frame | JPG/PNG |
| Banner horizontal | 1080 x 566 | 1.91:1 | JPG/PNG |

**Zona segura:** Manter 250px livres no topo e 340px livres na base (Stories) para nao cobrir elementos da interface.

**Recomendacao principal:** Usar **1080x1350 (4:5)** como formato padrao para feed. ocupa mais espaco na tela do celular e tem melhor taxa de parada de scroll.

## 12 Formatos de Imagem Estatica que Mais Convertem

### 1. TEXTO BOLD SOBRE FUNDO SOLIDO
- Fundo de cor unica (vibrante ou escuro)
- Texto grande e direto no centro
- Fonte sans-serif bold (Montserrat, Inter, Poppins)
- Maximo 7 palavras
- Funciona para: ganchos contraintuitivos, dados chocantes
- Exemplo: fundo preto, texto branco "97% dos cursos online nao ensinam isso"

### 2. BEFORE/AFTER (ANTES E DEPOIS)
- Dividido ao meio: lado esquerdo = antes, lado direito = depois
- Linha divisoria sutil ou barra central
- Funciona para: transformacao visual, resultados, comparativos
- Para infoprodutos: "antes de aplicar o metodo" vs "depois de 30 dias"
- Mostrar print de tela, numeros, metricas reais

### 3. UGC-STYLE (CONTEUDO GERADO POR USUARIO)
- Parece foto real tirada com celular
- Iluminacao natural, cenario do dia a dia
- Sem filtros pesados, sem look corporativo
- Texto pequeno no topo ou base como "legenda"
- Maior taxa de engajamento. parece organico, nao anuncio
- Para infoprodutos: pessoa segurando celular com o curso, print de resultado

### 4. SCREENSHOT DE CONVERSA
- Print de WhatsApp, DM ou comentario
- Mostra depoimento real ou pergunta frequente
- Borda do celular ao redor para contexto
- Texto acima: "Isso que acontece quando voce aplica o metodo"
- Alta credibilidade. prova social nativa

### 5. LISTA/CHECKLIST VISUAL
- 3 a 5 itens com icone de check ou numero
- Fundo limpo, tipografia clara
- Cada item e um beneficio ou etapa
- CTA no rodape: "Acesse o metodo completo"
- Funciona para: metodos, frameworks, passo a passo

### 6. DADO/ESTATISTICA DESTAQUE
- Numero grande e centralizado (ex: "83%", "R$47.000", "3x mais")
- Frase curta abaixo contextualizando o numero
- Fundo contrastante
- Fonte display ou serif para o numero
- Funciona para: resultados, provas, autoridade

### 7. MEME/HUMOR CONTEXTUAL
- Formato de meme classico com imagem + texto
- Humor relacionado a dor ou desejo do publico
- Tom leve mas conectado ao produto
- Gera compartilhamento e comentarios
- Para infoprodutos: situacoes que o publico vive no dia a dia

### 8. CARROSSEL EDUCATIVO (CAPA)
- Primeira imagem e a capa que para o scroll
- Titulo intrigante + "deslize para ver"
- Design limpo, 1 conceito por slide
- Ultima slide = CTA forte
- Funciona para: autoridade, educacao, nutrição

### 9. MOCKUP DE PRODUTO DIGITAL
- Imagem do produto em tela de notebook/tablet/celular
- Ambiente clean ou mesa de trabalho organizada
- Mostra "por dentro" do curso/ebook/ferramenta
- Badge de preco ou desconto no canto
- Para infoprodutos: gera tangibilidade de algo digital

### 10. QUOTE/FRASE DE AUTORIDADE
- Frase de impacto entre aspas
- Nome e credencial abaixo
- Foto pequena do autor ou icone
- Fundo neutro com tipografia elegante
- Funciona para: posicionamento, branding, confianca

### 11. PROBLEMA → SOLUCAO (SPLIT FRAME)
- Lado esquerdo: frustração, dor, situacao negativa
- Lado direito: solucao, resultado, situacao positiva
- Icones ou cores contrastantes (vermelho vs verde)
- Texto minimo. a imagem conta a historia
- Para infoprodutos: rotina atual vs rotina com o metodo

### 12. HEADLINE + FOTO LIFESTYLE
- Foto de alta qualidade no fundo (pessoa, cenario aspiracional)
- Headline bold no topo (maximo 10 palavras)
- Sub-headline menor abaixo
- CTA no rodape
- Formato classico de anuncio. funciona consistentemente

## Principios de Composicao que Param o Scroll

1. **Contraste extremo**. Texto claro sobre fundo escuro (ou vice-versa). Nada de cinza medio sobre cinza claro.
2. **Uma ideia por imagem**. Se precisa explicar, esta complexo demais.
3. **Hierarquia visual clara**. Olho vai primeiro para o elemento maior. Depois para o menor.
4. **Espaco vazio e respiro**. Nao preencher tudo. O vazio guia o olho.
5. **Rostos humanos**. Fotos com rosto tem 38% mais engajamento que objetos.
6. **Numeros especificos**. "R$47.231 em 14 dias" funciona melhor que "milhares de reais".
7. **Cores vibrantes**. Laranja, amarelo, vermelho e azul brilhante performam melhor no feed.
8. **Texto curto**. Maximo 20% da area da imagem com texto (regra do Meta Ads).

## Modelo de Prompt para Geracao de Imagem

Ao gerar prompts para a API de imagem, seguir esta estrutura:

```
[FORMATO]: Feed Instagram 1080x1350
[ESTILO]: [estilo escolhido do catalogo acima]
[COMPOSICAO]: [descricao da disposicao visual. centro, tercos, split]
[ELEMENTOS]: [lista dos elementos visuais. texto, foto, icone, badge]
[CORES]: [paleta principal. hex ou descritivo]
[TEXTO OVERLAY]: [texto exato que aparece na imagem]
[MOOD]: [sentimento geral. urgente, aspiracional, educativo, provocador]
[REFERENCIA]: [formato viral de referencia. ex: "Formato 4: Screenshot de Conversa"]
```

Converter para prompt em ingles otimizado para o modelo de IA (Flux, DALL-E, Stable Diffusion).

## Regras Especificas para Infoprodutos

1. **Nunca gerar rostos reais de pessoas**. Usar silhuetas, fotos de costas, maos, ou cenarios sem rosto identificavel
2. **Mostrar o produto tangivel**. Mockup de tela, print de modulo, imagem do material
3. **Priorizar resultado sobre processo**. Imagem do "depois", nao do "durante"
4. **Incluir elemento de prova**. Numero, print, badge de garantia
5. **CTA visual**. Botao ou seta apontando para a acao (mesmo que decorativo)
