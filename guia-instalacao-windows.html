Fluxo Criativo
Guia de instalação para Windows
⚠️
Por que o Windows mostra um aviso?
O Windows bloqueia apps que ainda não têm certificado de publicadora registrado. O Fluxo Criativo é seguro. Siga os passos abaixo para instalar sem problemas.
Opção 1 — Instalador (.exe)
1
Abra o instalador
Dê dois cliques no arquivo Workshop-IA-Setup.exe. O Windows vai mostrar a tela abaixo.
Windows protegeu seu computador
🛡️
O Windows Defender SmartScreen impediu
Um aplicativo não reconhecido pode colocar seu PC em risco.
👆 Clique em "Mais informações"
2
Clique em "Mais informações"
A tela vai atualizar e mostrar o botão "Executar mesmo assim".
Windows protegeu seu computador
🛡️
O Windows Defender SmartScreen impediu
Aplicativo: Workshop-IA-Setup.exe  |  Publicador: desconhecido
👆 Clique em "Executar mesmo assim"
3
Instalação concluída
O instalador vai abrir, concluir em segundos e criar um atalho na área de trabalho. Abra o Fluxo Criativo pelo atalho.
Opção 2 — ZIP portátil (sem instalador)
Prefere não usar instalador?
Baixe o arquivo Workshop-IA-win-x64.zip na aba de downloads, extraia em qualquer pasta e abra o Fluxo Criativo.exe diretamente. O Windows ainda pode mostrar o aviso, mas o processo é mais simples: basta clicar em "Mais informações" → "Executar mesmo assim" uma única vez.
✅
Pronto, pode usar.
Esse aviso aparece apenas na primeira vez. Nas próximas aberturas o Fluxo Criativo abre normalmente.
